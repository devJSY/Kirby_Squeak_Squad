# 📍 Operation key

- Move : ←, →
- Release Transformations - W
- Jump - A or D
- Skill - S
- Swallow  - ↓ 
- Portal - ↑
- Inventory - Mouse
- Level Unlock - KeyPad 1 ~ 9
- Level Clear Unlock, Pixel Texture Render - T
- Add Random AbilityItem - R
- Add Recovery item - H

- Skip Scene - MOUSE_MBTN or A or D

# 📍 How to execute

1. `<> Code ▼` Click
2. Download ZIP
3. Unzip
4. Release → Client.exe execute
